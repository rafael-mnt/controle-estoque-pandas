# Sistema de Controle de Estoque
Ferramenta para gerenciamento de estoque desenvolvida em Python, utilizando a biblioteca Pandas para manipulação de dados em planilhas Excel.

## 💻 Funcionalidades
- Cadastro de produtos
- Remoção de produtos
- Atualização de estoque
- Consulta de itens cadastrados
- Validação de entradas do usuário
- Integração com arquivos Excel

## ⚙️ Tecnologias Utilizadas
- Python
- Pandas
- Openpyxl

## 💡Aprendizados
Durante o desenvolvimento deste projeto, foram praticados conceitos como:

- Manipulação de dados com Pandas
- Estruturação de código em funções
- Validação de códigos de entrada
- Organização de lógica de sistema simples
